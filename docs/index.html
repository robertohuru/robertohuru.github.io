Accessibility · GeoComputation · Algorithms 📚

R TravelCost Package

R package that computes travel cost for each pixel in a friction surface to a target point. It calculates weighed cumulative distance of each grid cell to the nearest point in a set of target points or grid cells based on a weight grid. Multiple target locations are processed at once by using a two-phase concurrent implementation of Dijkstra's algorithm. The core of the algorithm is built in C++ and is designed to be fast and memory efficient thereby reducing processing runtime by 65 - 90% on large raster datasets when compared to related packages.

R C++
